## Tools
- Backend: Spring Framework(Legacy) - Security, WebMVC, Web, Data JPA, Transaction, JDBC
- Frontend: Vue.js - Router, Vuex, + BootStrap/Vanilla CSS
- Network: Axios, Vanilla Ajax
- DataBase: Oracle XE 11g
- Server: Tomcat 9.0
